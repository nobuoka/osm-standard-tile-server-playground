## How to use

```
terraform init
```

Set up environment variables in bash:

```
export AWS_ACCESS_KEY_ID="YOUR_KEY"
export AWS_SECRET_ACCESS_KEY="YOUR_SECRET_KEY"
```

Or in PowerShell:

```
$env:AWS_ACCESS_KEY_ID="YOUR_KEY"
$env:AWS_SECRET_ACCESS_KEY="YOUR_SECRET_KEY"
```

```
terraform apply
```
